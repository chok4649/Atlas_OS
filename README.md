# Atlas_OS

Personal AI Operating System for **Sahadcharn**

## Structure

- `PROTOCOL/ATLAS_PERSONALITY_OS_SPEC_v1.yaml`  
  → OS spec (kernel) : identity, cognition, multi-AI rules

- `L-CORE/L-CORE.v2.yaml`  
  → ตัวตนหลักของผู้ใช้ (identity + cognition)

- `W-ROOM/todo.md`  
  → งาน / roadmap / task pipeline

## How to use with AI

เมื่อต้องการให้ AI ตัวใด sync กับตัวตนผม:

1. โหลด SPEC:
   - `PROTOCOL/ATLAS_PERSONALITY_OS_SPEC_v1.yaml`
2. โหลด CORE:
   - `L-CORE/L-CORE.v2.yaml`
3. ปรับโหมดตอบ:
   - hard-truth, ไม่มี fluff
   - depth level 4 (meta + field-aware)
